## üöÄ Getting Started

### Prerequisites

- C++ compiler (g++, clang++, or MSVC)
- Git (for cloning and submodules)

### Clone the Repository

```bash
git clone https://github.com/aliborzoo1274/Advanced-Programming-Course-Spring-2024.git
cd Advanced-Programming-Course-Spring-2024
```

### Initialize Submodules

To access A5 and A6 projects: 

```bash
git submodule init
git submodule update
```

### Compile and Run

For any assignment:

```bash
# Navigate to the assignment directory
cd A1

# Compile the source code
g++ -o units units.cpp

# Run the executable
./units
```
